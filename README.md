## 贡献指南
请在issue中提出你购买的书籍，格式如下
```bash
issue-name:
    《书名》 - 作者或者编著之一二
issue-body:
    《书名》 - (版本/作者)
    书籍简介
    推荐✔/不推荐❌
    简单阐述推荐✔/不推荐❌的理由
    一张图书的照片
    * 晒一下购买价格   
    
    # * 号为可选项，也可以参考我提交的issue😄
```
根据issue不断更新项目内容
